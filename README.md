# 🏆 School Tournament Draw

A **fully client-side, zero-dependency** web app for running live draws and generating brackets for school sports tournaments.

Supports **Soccer ⚽**, **Basketball 🏀**, and **Volleyball 🏐** — each with independent bracket generation.

## ✨ Features

- 🎲 **Live draw** — pick teams from urns with animated chip selection
- 🏷️ **Two-group mode** — split teams into Group A and Group B, then merge into a single bracket
- 🖼️ **SVG bracket** — auto-generated tournament bracket, updates in real time
- ⛔ **BYE support** — handles odd number of teams automatically
- 🌑 **Dark UI** — sleek dark theme, fully responsive
- 📱 **No install needed** — pure HTML/CSS/JS, just open the file

## 🚀 Live Demo

👉 [Open the app](https://YOUR-USERNAME.github.io/school-tournament-draw/)

## 🛠️ How to Use

1. Select a sport tab (Soccer / Basketball / Volleyball)
2. Enter class/team names (one per line)
3. Optionally enable **two-group mode**
4. Click **Carica Classi** to start the draw
5. Click chips to select two teams → **Conferma Coppia**
6. The bracket updates live on the right panel

## 🧰 Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JavaScript**
- **SVG** for bracket rendering
- No frameworks, no build step, no dependencies
